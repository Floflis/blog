I’ve scanned bin, etc, lib, lib64, opt, root, sbin, srv, usr, var root directories, which will be used by live-linux scriptset to build the ISO. 17.4Â GB is very challenging, as the maximum threshold I’m defining for the distribution .ISO is 3,5 GB.
